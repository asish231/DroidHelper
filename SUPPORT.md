# Support

## Getting Help

Use GitHub issues for:

- bug reports
- feature requests
- troubleshooting questions

## Before Opening an Issue

- read the `README.md`
- confirm whether the issue is in USB mode or wireless mode
- include your OS, phone model, and Android version
- include the exact error output when possible

## What This Project Supports

DroidHelper supports:

- guided `adb` and `scrcpy` setup
- USB launch flow
- wireless pairing and reconnect flow
- audio mode selection

This project does not replace upstream support for:

- `scrcpy`
- Android platform tools
- device-specific Android ROM restrictions

## Response Expectations

This is an open-source project and support is best-effort.
