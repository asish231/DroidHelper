# Code of Conduct

## Our Standard

This project aims to be practical, respectful, and useful to contributors and users.

Participants are expected to:

- be respectful in discussions and review comments
- assume good intent, then clarify with facts
- give actionable technical feedback
- avoid harassment, insults, discrimination, and personal attacks

## Unacceptable Behavior

Examples of unacceptable behavior include:

- personal attacks or abusive language
- harassment or intimidation
- discrimination based on personal identity or background
- repeated bad-faith disruption of discussion or review
- sharing private information without permission

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, pull requests, or other contributions that violate this code of conduct.

Serious or repeated violations may result in temporary or permanent exclusion from the project spaces.

## Reporting

If you see a problem, open a private report to the repository owner through GitHub or open an issue if the matter does not require privacy.

## Scope

This code of conduct applies to issues, pull requests, discussions, and other public interactions in this repository.
